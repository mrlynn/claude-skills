## Example: RAG Systems That Don't Hallucinate

## Abstract

RAG systems promise accurate AI responses, but 60% of production implementations still hallucinate. Why?

In this talk, you'll learn the 5 patterns that make RAG systems trustworthy: content hashing for incremental updates, category-based score boosting, hybrid search (vector + keyword), source attribution, and feedback loops. We'll build a production RAG system live, handling 10M documents with <200ms query latency using MongoDB Atlas Vector Search and Voyage AI embeddings.

**You'll learn:**
- Incremental ingestion pattern (90% cost savings)
- Score boosting for retrieval quality (40% improvement)
- Production monitoring with MRR and Precision@K metrics
- Cost optimization for embeddings and storage

**Target audience:** Backend developers building AI-powered applications with Python

**Speaker experience:** Built RAG systems for 5 Fortune 500 companies, processing 50M+ embeddings. Sharing production patterns and gotchas from 20+ deployments.
